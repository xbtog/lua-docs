# json

## Functions

### parse

| Argument | Type           | Required |
| -------- | -------------- | -------- |
| data     | `string{:lua}` | +        |
| returns  | `table{:lua}`  |          |

```lua
local parsed = json.parse("{\"hello\": \"world\"}");

print(parsed["hello"]) -- prints: 'world'
```

### stringify

| Argument | Type           | Required |
| -------- | -------------- | -------- |
| data     | `table{:lua}`  | +        |
| returns  | `string{:lua}` |          |

```lua
local str = json.stringify({hello = "world"});

print(str) -- prints: '{"hello":"world"}'
```
