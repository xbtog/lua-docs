# helper

## Functions

### calculate_damage

Calculates the actual damage when taking armor / magic resist into account.

| Argument        | Type      | Required |
| --------------- | --------- | -------- |
| raw_damage      | `number`  | +        |
| target_index    | `number`  | +        |
| physical_damage | `boolean` | +        |
| returns         | `number`  |          |

### get_aa_damage

Get local player next autoattack damage.

| Argument     | Type      | Required |
| ------------ | --------- | -------- |
| target_index | `number`  | +        |
| on_hit       | `boolean` | +        |
| returns      | `number`  |          |

### get_on_hit_damage

Get next autoattack/spell onhit damage.

| Argument     | Type     | Required |
| ------------ | -------- | -------- |
| target_index | `number` | +        |
| returns      | `number` |          |

### get_real_health 

| Argument     | Type     | Required |
| ------------ | -------- | -------- |
| target_index | `number` | +        |
| damage_type | `number` | +        |
| delay | `number` | +        |
| predict_ally_damage | `boolean` | +        |
| returns      | `number` |          |

--- damage_type = 0 -> true_damage
