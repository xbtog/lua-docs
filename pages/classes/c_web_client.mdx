# c_web_client

## Functions

### get

| Argument | Type                                                                                                             | Required |
| -------- | ---------------------------------------------------------------------------------------------------------------- | -------- |
| url      | `string{:lua}`                                                                                                   |          |
| options  | `{timeout: number \| nil, body: string \| table \| nil, useragent: string \| nil, headers: table \| nil }{:lua}` |          |
| returns  | [`c_response`](/classes/c_web_client/c_response)                                                                 |          |

```lua
local web_client = c_web_client.new()

local response = web_client:get(
    "https://example.com",
    {
        useragent = "useragent",
        headers = {{"content-type", "application/json"}},
        timeout = 30 --[[ 30 seconds --]]
    }
)

local json_table = response:json()
```

### async_get

| Argument | Type                                                                                                             | Required |
| -------- | ---------------------------------------------------------------------------------------------------------------- | -------- |
| url      | `string{:lua}`                                                                                                   |          |
| options  | `{timeout: number \| nil, body: string \| table \| nil, useragent: string \| nil, headers: table \| nil }{:lua}` |          |
| callback | `function(response: c_response){:lua}`                                                                           |          |
| returns  | [`c_response`](/classes/c_web_client/c_response)                                                                 |          |

```lua
local web_client = c_web_client.new()

web_client:async_get("https://example.com", {}, function(response)

end)
```
