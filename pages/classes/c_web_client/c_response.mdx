# c_response

## Members

- `success:` `boolean`
- `status:` `number`

## Functions

### json

Parse the response as JSON.

```lua
local json = response:json()

print(json["error"])
```

### text

Get the response as text.

```lua
local text = response:text()
```

### to_file

Save response text to file.

```lua
response:to_file("C:\\test.txt")
```
