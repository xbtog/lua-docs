# c_entity_list

## Functions

### get_enemies

Get a list of all enemy champions.

| Argument | Type                                         | Required |
| -------- | -------------------------------------------- | -------- |
| returns  | [`table<c_object>{:lua}`](/classes/c_object) |          |

```lua /get_enemies/
local enemies = features.entity_list:get_enemies()

-- print all enemy champion names
for _, enemy in pairs(enemies) do
    print(enem.champion_name.text)
end
```
