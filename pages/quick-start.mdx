## Getting the LUA SDK

You can download the SDK for your LUA api [here](https://github.com/xbtog/lua-sdk) for ide autocompletion.

## Registering a callback

```lua
cheat.on("renderer.draw", function()
    print("hello from render callback")
end)
```

## Loading LUA libraries

```lua
-- example
-- loads "extra_math" fron <lua_folder>/lib
local extra_math = require("extra_math")
```
